# Data Privacy Review

## Goal

Verify that the system collects, uses, retains, shares, exports, and deletes only the data it legitimately needs.

Read [references/data-privacy-checklist.md](references/data-privacy-checklist.md) for the detailed data-flow and lifecycle review.

## Workflow

1. Inventory data fields, sources, purposes, owners, classifications, and affected users.
2. Map storage, logs, analytics, exports, backups, third parties, AI providers, and cross-boundary transfers.
3. Verify minimization, consent or lawful basis, access controls, retention, deletion, correction, and export behavior.
4. Verify masking, encryption, redaction, and safe test data.
5. Record privacy risks, owners, required changes, and residual risk.

## Output

Report data inventory, purpose, flows, third parties, retention, deletion, user controls, protection, findings, required evidence, and residual risk.
