# Good Security Review: Authorized Scope

## Authorization

- Written approval linked.
- Staging environment and approved test accounts identified.
- Production, third-party systems, and real user data excluded.
- Request-rate limits, monitoring contacts, stop conditions, and evidence rules recorded.

## Review Plan

- Begin with architecture, code, configuration, and dependency review.
- Use controlled active validation only for approved assets.
- Stop after the minimum evidence needed to confirm impact.
